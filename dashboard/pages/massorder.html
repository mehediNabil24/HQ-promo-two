How Mass Order works?

You put the service ID followed by | followed by the link followed by | followed by quantity on each line To get the service ID of a service please check here:

 

https://hqpromo.com/services

 

Let’s say you want to use the Mass Order to add Instagram Followers to your 3 accounts: abcd, asdf, qwer From the Services List, the service ID for this service “[Instagram Followers [100% Real - 30 Days Guarantee- NEW SERVICE” is 3740

 

Let’s say you want to add 1000 followers for each account, the output will be like this: ID|Link|Quantity.
or in this example:

  • 3740|abcd|1000
  • 3740|abcd|1000
  • 3740|abcd|1000
  • 3740|abcd|1000
